	\begin{block}{Why}
		Bag of words models are only so good. Wouldn't it be nice if we could search semantically?
	\end{block}
	\begin{block}{What}

		Entities ``Michael Jackson", ``Motown Records", ``The Jackson 5ive"

\end{frame}
